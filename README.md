# 🔐 Smart Security & Autonomous Mobility System

An integrated mechatronics project featuring biometric authentication, LiDAR-based obstacle avoidance, and precision weight sensing, controlled via a unified MCU architecture.

## 🎯 Project Overview
This project involves the design and implementation of an autonomous security system. It combines multiple sensor modalities—including fingerprint scanning, LiDAR distance measurement, and load cell weight detection—to create a responsive and secure robotic platform.

## 🛠️ Key Technical Implementations

* **Biometric Authentication:** Integrated a fingerprint sensor for secure user access control, utilizing high-confidence matching algorithms.
* **Autonomous Navigation:** Implemented **LiDAR-based distance measurement** to enable real-time obstacle detection and safe movement protocols.
* **Precision Weight Sensing:** Developed a load cell system using the **Wheatstone Bridge principle** and the NAU7802 24-bit ADC for high-accuracy mass measurement.
* **Motor Kinematics & PWM:** Designed a motor control system utilizing **Pulse Width Modulation (PWM)** to achieve precise velocity and directional control.
* **System Architecture:** Engineered the complete electrical circuit, including power management for MCUs, sensors, and high-torque actuators.
